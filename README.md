# WCAG Code A11y Agent

An MCP (Model Context Protocol) agent that analyzes project files and related modules to determine relevant WCAG (Web Content Accessibility Guidelines) Success Criteria for developers and QA teams.

## What It Does

- Analyzes code files (React, Vue, Angular, HTML, etc.)

- Examines related modules to understand component context

- Suggests WCAG SC that might apply to the component

- Helps developers build accessible code from the start

- Assists QA teams in accessibility testing
